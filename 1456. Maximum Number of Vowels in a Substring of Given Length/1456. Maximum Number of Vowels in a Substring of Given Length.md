# 1456. Maximum Number of Vowels in a Substring of Given Length

**Difficulty:** Medium  
**LeetCode:** https://leetcode.com/problems/maximum-number-of-vowels-in-a-substring-of-given-length/

---

## Problem Statement

Given a string s and an integer k, return the maximum number of vowel letters in any substring of s with length k.

Vowel letters in English are 'a', 'e', 'i', 'o', and 'u'.

Example 1:
Input: s = "abciiidef", k = 3
Output: 3
Explanation: The substring "iii" contains 3 vowel letters.

Example 2:
Input: s = "aeiou", k = 2
Output: 2
Explanation: Any substring of length 2 contains 2 vowels.

Example 3:
Input: s = "leetcode", k = 3
Output: 2
Explanation: "lee", "eet" and "ode" contain 2 vowels.

Constraints:
1 <= s.length <= 10^5
s consists of lowercase English letters.
1 <= k <= s.length

---

## Solution 1: Sliding Window with Vowel Map

```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        vowel_map = {'a': 1, 'e': 1, 'i': 1, 'o': 1, 'u': 1}
        outgoing_vowel = 0
        current_count = 0
        max_count = 0

        for i in range(len(s)):
            current_count -= outgoing_vowel
            current_count += vowel_map.get(s[i], 0)
            max_count = max(max_count, current_count)
            if i >= k - 1:
                outgoing_vowel = vowel_map.get(s[i - k + 1], 0)
        return max_count
```

### Complexity

| | Complexity | Notes |
|---|---|---|
| **Time** | O(n) | single pass through the string of length n |
| **Space** | O(1) | vowel_map has constant size, only a few integer variables |
